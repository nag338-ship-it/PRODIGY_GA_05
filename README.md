# PRODIGY_GA_05

## Neural Style Transfer

### 📌 Task
Apply the artistic style of one image to the content 
of another image using Neural Style Transfer.

### 🛠️ Tools & Libraries Used
- Python
- PyTorch
- torchvision (VGG19)
- Matplotlib
- PIL
- Google Colab (T4 GPU)

### 🏗️ How It Works
- Content Image: A regular photo
- Style Image: Van Gogh's Starry Night painting
- Model: Pretrained VGG19 extracts features
- Optimization: LBFGS optimizer minimizes combined loss
- Output: Photo with Van Gogh's artistic style!

### 📊 Loss Functions
- Content Loss: MSE between content features
- Style Loss: Gram matrix difference of style features
